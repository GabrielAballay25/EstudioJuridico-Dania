# ⚖️ Estudio Jurídico Dania Villanueva

Landing page institucional desarrollada para la Dra. Dania Villanueva, abogada. El sitio está pensado para transmitir profesionalismo y facilitar el contacto directo de potenciales clientes con el estudio.

🔗 **Sitio en producción:** [abogadavillanueva.com.ar](https://abogadavillanueva.com.ar)

## ✨ Características

- **100% Responsive** — Experiencia optimizada en mobile, tablet y desktop.
- **Animaciones fluidas** con Framer Motion para una navegación más dinámica y moderna.
- **Integración con WhatsApp** — Contacto directo con un solo clic, sin fricción para el usuario.
- **Visor de PDF** integrado para mostrar documentación / información institucional.
- **Carrusel de contenido** para destacar áreas de práctica o testimonios.

## 🛠️ Stack tecnológico

| Tecnología | Uso |
|---|---|
| React 19 | Librería principal de UI |
| Vite | Bundler y entorno de desarrollo |
| Bootstrap / react-bootstrap | Sistema de grillas y componentes base |
| Framer Motion | Animaciones e interacciones |
| react-slick / slick-carousel | Carrusel de contenido |
| react-pdf | Visualización de documentos PDF |
| lucide-react / react-icons | Iconografía |
| ESLint | Linting y calidad de código |

## 🚀 Cómo correr el proyecto localmente

```bash
# Clonar el repositorio
git clone https://github.com/GabrielAballay25/EstudioJuridico-Dania.git

# Ingresar a la carpeta del frontend
cd EstudioJuridico-Dania/front-estudio-juridico-dania

# Instalar dependencias
npm install

# Correr en modo desarrollo
npm run dev
```
